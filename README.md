# 保研加分助手
厦门大学交互设计课程作业。<br/>
前端使用Vue，后端使用Flask，数据库使用SQLite。<br/>

## 第一阶段需要实现：
1.学生提交材料（已实现）<br/>
2.学生查看已提交材料的列表（已实现）<br/>
3.教师查看所有学生提交列表（已实现）<br/>
4.教师修改材料审核状态（已实现）<br/>
5.登录（已实现）<br/>

## 第二阶段需要实现：
1.已提交材料筛选器与排序功能（已实现）<br/>
2.计算审核中的加分、已通过的加分并显示在学生页面上（已实现）<br/>
3.学生提交材料时可以附加详细信息，并选择提交材料类型<br/>
4.教师驳回申请时可以选择驳回理由，如格式不符、分数错误、图片模糊等（已实现）<br/>

## 第三阶段需要实现：
1.美化ui（已实现）<br/>
2.生成汇总表格，使其可以复制到excel中<br/>
